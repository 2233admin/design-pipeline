# Directions

## Selected: Stable contracts, optional adapters

Schemas and deterministic validators define portable artifacts. A unified CLI delegates to kernels.
Adapters execute only from explicit trusted local paths and return receipts.

## Rejected

- Framework-first expansion: too much duplicated, version-sensitive behavior.
- Documentation-only contracts: cannot prove Agent output or runtime evidence.
- Bundled browser/design SDKs: violates offline and replacement boundaries.
