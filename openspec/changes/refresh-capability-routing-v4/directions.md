# Directions

## A. Keep static folder checks

Smallest change, but it preserves the false-green problem and cannot express version-sensitive runtime capabilities.

## B. Capability-first routing with bounded profiles — selected

Keep the current dependency-free self-check, add a routing reference, and introduce one marker-based Anime.js v4.5 profile. This improves decisions without turning the package into a marketplace or package manager.

## C. Crawl every plugin and skill cache

Would discover more host surfaces automatically, but couples the standalone package to unstable runtime layouts and expands the security/maintenance surface.

## Decision

Use direction B. Runtime/plugin hosts may add explicit adapters later; the standalone skill only accepts explicit skill roots.
