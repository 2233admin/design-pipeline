# Proposal: add contribution standards

## Why

The repository is intended to be open source. Contributors need clear rules for design-first scope, external skill intake, QA, security, and OpenSpec-aligned changes.

## What

- Add contribution guidelines.
- Add code of conduct.
- Add security guidance.
- Add GitHub issue and PR templates.
- Add an OpenSpec delta for contribution governance.

## Non-Goals

- Do not create a large governance framework.
- Do not require optional companion skills for contribution.
